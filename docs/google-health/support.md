# Google Health API - Support

## Overview

The Google Health API support page provides resources for developers to report issues, get help, and connect with the community.

## Support Resources

### [Issue Tracker](https://issuetracker.google.com/issues/new?component=1914241)

Report issues or send feedback to Google's development team. Include detailed information about your issue along with supporting files such as logs and videos to help the team reproduce and troubleshoot problems.

### [Developer Forum](https://community.fitbit.com/t5/Web-API-Development/bd-p/dev)

Connect with the developer community to discuss Web API development topics.

### [Fitbit Help Center](http://support.google.com/fitbit)

Access resources about Fitbit devices and get answers from community members.

## Important Timeline Notice

"Start your migration or new app development today!" Developers should plan for official launch integration by the **end of May 2026** to align with legacy Fitbit account deprecation. Breaking changes may occur until the end of May as feedback is incorporated.
