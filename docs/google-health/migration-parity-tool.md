# Google Health API Parity Tool

## Overview

The Google Health API parity tool assists developers migrating from the Fitbit Web API. It enables comparison of endpoint functionality and responses between the two platforms.

## Migration Timeline

"To ensure a seamless experience for your users, we recommend waiting until the **end of May 2026** to officially launch your integration" to align with legacy Fitbit account deprecation. Developers should note that breaking changes may occur through May as feedback is addressed.
